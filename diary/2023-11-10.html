Vimwiki Markdown Tag Syntax

pandoc Obsidian vimwiki tags vimwiki_tags

  • Convert Vimwiki Format to Obsidian Markdown with Pandoc developed a Makefile to convert Vimwiki native format to Markdown and convert the tags to #tag style.
  • Partially working solution. VimwikiRebuildTags fails. 
    let g:vimwiki_syntaxlocal_vars.markdown.tag_format = {
  \ 'pre_mark': '\%(:\)',
  \ 'sep': '\%(:\)',
  \ 'cchar': '',
  \ 'post_mark': '\%(:\)',
  \ 'conceal': 0,
  \ 'post': '\%(\s\|$\)',
  \ 'in': '\%([^:''[:space:]]\+\)',
  \ 'pre': '\%(^\|\s\)'
  \ }
