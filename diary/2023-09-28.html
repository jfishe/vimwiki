Zotero Zettelkasten

Retorquere / zotero-deb provides a Debian ppa to install Zotero per zotero.org 

wget -qO- \
  https://raw.githubusercontent.com/retorquere/zotero-deb/master/install.sh |
  sudo bash
sudo apt update
sudo apt install zotero
